 🎥Live Object Detection Web App

A real-time object detection web application built using:
* Python
* Streamlit
* YOLOv8
* OpenCV
* Streamlit WebRTC
* MediaPipe

The application allows users to detect objects live through a webcam directly in the browser.

🎯 Features
* Real-time webcam object detection
* YOLOv8 integration
* Browser-based camera streaming
* Fast and responsive interface
* MediaPipe support
* Streamlit web deployment
* Clean and interactive UI

🛠️ Technologies Used

* Python
* Streamlit
* YOLOv8
* OpenCV
* MediaPipe
* Streamlit-WebRTC
* PyTorch

💼 How It Works

1. The webcam stream is captured using Streamlit WebRTC.
2. Frames are processed using OpenCV.
3. YOLOv8 detects objects in real time.
4. Bounding boxes and labels are displayed on the video feed.
5. The processed stream is shown directly in the browser.
